# 使用方式
## 連結CDN

```
<script src="https://bonnie19931112.github.io/ScrollToElemet/main.js"></script>

```

## 捲動說明
屬性名稱 | 屬性說明
--------|--------
data-st-target | 要前往 section 的ID名稱
data-st-duration | 捲動所花的時間(毫秒
data-st-offset | 位移的大小

## 箭頭說明
返回頂端箭頭必須添加 ID arrow

屬性名稱 | 屬性說明
--------|--------
data-st-top | 箭頭出現位置
data-st-time | 箭頭的持續時間
